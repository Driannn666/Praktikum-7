1. Alur Flowchart Pengerjaan

Program dimulai dari fungsi main().
Membuat objek mhs1 dari class Mahasiswa.
Mengisi atribut (nama, umur, alamat).
Menampilkan data menggunakan method class Person dan Mahasiswa.
Membuat objek mahasiswa lain (mhs2).
Menampilkan informasi lengkap kedua mahasiswa.
Program selesai.

2. Cara Menjalankan Program

Pastikan Python sudah terinstal.
Buka file praktikum7.py di VSCode atau editor lain.
Jalankan program dengan perintah:
python praktikum7.py
Output akan muncul di terminal.
